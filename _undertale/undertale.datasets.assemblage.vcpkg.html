vcpkg¶

The VCPKG dataset, built by Assemblage.

VCPKG (https://vcpkg.io/en/) is an open-source library manager for downloading and building external libraries. This dataset was built by the Assemblage folks who downloaded and compiled all of VCPKG.

Data: https://assemblage-dataset.net/
